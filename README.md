# How to make changes to this project

```bash
# 1. change to the repo directory
cd ~/Documents/kent/60692site/gardinerjw.github.io
# 2. open atom (if you haven't already)
atom .
# 3. run the jekyll dev server
bundle exec jekyll server
# 4. make changes in atom ... and preview changes at http://localhost:4000
# 5. build static site
bundle exec jekyll build
# 6. commit changes in github for mac
# 7. sync changes in github for mac
# 8. check https://gardinerjw.github.io
# 9. control+C in Terminal to end process at end of session
```
